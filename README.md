# IOS-Swift-SlideInMenu
``` Level : Intermediate ```

In this tutorial, Learn the easy way to Create Slide in effect Project to show slide in menu from any side you wish included right, left , top and bottom in Swift with different cool UI in Xcode by Swift Program (NSObject, UICollectionView ) You will be using NSObject, UICollectionView, UICollectionViewDataSource, UICollectionViewDelegate, addConsTraintsWithFormat, NSLayoutConstraint and understand how to add Constraints programmatically by swift. 

## About

Sample app to add Ready SlideIn Menu to Your ViewControler in Few Minutes ..

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You just add one swift file to your prject .
